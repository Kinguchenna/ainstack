1. Generate Audio from Text

POST /api/v1/text-to-audio

Request Body

Parameter Type Description
text string The text to convert into speech
voice string Voice style (e.g., 'male', 'female', 'narrator')
language string Language code (e.g., 'en-US')

Sample Request

{
  "text": "Welcome to Eldoria, land of dragons.",
  "voice": "narrator",
  "language": "en-US"
}

Sample Response

{
  "audio_url": "https://yourdomain.com/audio/eldoria.mp3",
  "status": "success"
}

2. Transcribe Audio to Text

POST /api/v1/audio-to-text

Request Body

Parameter Type Description
file file (audio) Upload MP3 or WAV audio file
language string Language of the spoken content (e.g., 'en-US')

Sample Response

{
  "transcript": "Welcome to Eldoria, land of dragons.",
  "confidence": 0.98,
  "status": "success"
}

3. Convert Text to Song

POST /api/v1/text-to-song

Request Body

Parameter Type Description
lyrics string Text or lyrics to convert into a song
genre string Style or genre (e.g., 'pop', 'rock', 'lofi')

Sample Response

{
  "song_url": "https://yourdomain.com/songs/my_song.mp3",
  "status": "success"
}
